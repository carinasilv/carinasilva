Ruby Period

App Design · Figma Prototype · 2026

Developed for Value Creation course at Arizona State University. Ruby Period is inspired by my own experience with PMDD, designed to support women with menstrual disorders by providing access to help, data, and insights into their cycles.

Process

Paired with Principles of User Interface Design, this project had two main focuses: researching the target audience and creating value for a specific segment. The interface was built in Figma using core UI principles, resulting in a working clickable prototype. Of of 100 student projects, My pitch was voted into the top 10 earning the opportunity to receive $1,000 in funding through Arizona State University W.P. Carey Business School
